# Stripe Atlas Brain

> "Stripe Atlas saved me when I started my first business at 19 — not the incorporation, but the guides." — Annissa Poon

Stripe Atlas has helped 100,000+ companies get started. But the best part isn't the incorporation — it's the founder education content that's been buried for years.

**This repo makes it accessible.**

Stripe Atlas Brain turns Stripe's founder education into AI-native skills you can plug into Claude Code, Cursor, or any LLM that supports the SKILL.md standard. Get Patrick Collison, Michael Seibel, and YC-level thinking on demand.

## What's included

| Skill | What it does |
|-------|--------------|
| `/atlas-office-hours` | Founder office hours — hard questions before you build |
| `/pitch` | Nail your investor pitch (built with YC's Michael Seibel) |
| `/pitch-deck` | Structure your seed/Series A deck using the 5 pitch plots |
| `/equity` | Understand founder equity, vesting, dilution |
| `/hiring` | Recruit and manage early talent |
| `/founder-stress` | Manage the psychological load of building |

## Quick start

1. Install (30 seconds):
```bash
git clone https://github.com/annissapoon/stripe-atlas-brain.git ~/.claude/skills/stripe-atlas-brain
```

2. In Claude Code:
```
/atlas-office-hours "I'm building a marketplace for spiritual guidance"
```

3. Follow the prompts. Get challenged. Build something real.
